||strings, string|

||numbers, number|

# Example Codes
**hello_world.fl :**
---------------------
```rust
fn main() {
    out("Hello World\n", output.txt);
}
```
**one\_to\_ten.fl:**
---------------------
```rust
fn main() {
    for(let i = 1; i <= 10; i++) {
        out(i, output.txt);
    }
}
```
**fibonacci.fl:**
---------------------
```rust
fn fibonacci(n) {
    if(n <= 2) {
        return 1;
    }
    fibonacci(n - 1) + fibonacci(n - 2)
}
```
